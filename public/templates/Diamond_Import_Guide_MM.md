## ၁. Template ဖိုင်ကိုဒေါင်းလုပ် ရယူပါ
- [Diamond Products Import Template ကိုဒေါင်းလုပ် ရယူရန်](/templates/diamond_products_import_template_final.csv)

## ၂. Excel ဖိုင်ဖြည့်စည်းနည်း

### အဓိက အချက်များ
- **no**: ထုတ်ကုန်အမှတ်စဥ် (ဥပမာ - DIAM-001)
- **item_name**: ပစ္စည်းအမည် (မြန်မာလို ရေးပါ)
- **gold_quality**: ရွှေအရည်အသွေး (18K, 14K)
- **sale_fixed_price**: ရောင်းစျေး (ကျပ်)
- **original_fixed_price**: ကုန်ကျစရိတ် (ကျပ်)

### အသေးစိတ်ဖြည့်စည်းနည်း
1. **အခြေခံအချက်အလက်**
   - `staff_name`: ဝန်ထမ်းအမည်
   - `goldsmith_name`: ပန်းထိမ်သည်အမည်
   - `supplier`: ပေးသွင်းသူအမည်

2. **ပစ္စည်းအချက်အလက်**
   - `length`, `width`: အလျား၊ အနံ (မီလီမီတာ)
   - `color`: အရောင် (ဥပမာ - ဖြူ၊ ရွှေ၊ ပန်းရောင်)
   - `remark`: အထူးမှတ်ချက်

3. **စျေးနှုန်းနှင့်ကုန်ကျစရိတ်**
   - `design_charges`: ဒီဇိုင်းခ
   - `plating_charges`: ရွှေချွတ်ခ
   - `mounting_charges`: စိန်တပ်ခ
   - `white_charges`: အဖြူရောင်ချွတ်ခ
   - `other_charges`: အခြားကုန်ကျစရိတ်

## ၃. Import လုပ်နည်း
1. Admin Panel သို့ဝင်ပါ
2. Products ကဏ္ဍတွင် "Import" ကိုနှိပ်ပါ
3. ဖြည့်စည်းပြီးသော CSV ဖိုင်ကိုရွေးပါ
4. Upload လုပ်ရန် ခလုတ်ကိုနှိပ်ပါ
5. Import အောင်မြင်ကြောင်း အတည်ပြုချက်ကိုစောင့်ပါ

## အရေးကြီးသောမှတ်ချက်များ
- ဖိုင်ကို CSV (UTF-8) format ဖြင့်သာသိမ်းပါ
- အကွက်လွတ်များကို 0 ဖြင့်ဖြည့်ပါ
- ငွေကြေးပမာဏများကို ဒဿမမပါဘဲ ရေးပါ (ဥပမာ - 500000)
